Desafio Técnico

O sistema deve validar os campos obrigatórios (nome, CPF, data de nascimento, e-mail, e CEP), consultar a API ViaCEP para preencher dados de endereço, e exibir as informações cadastradas em uma tabela. O cadastro deve ser realizado de forma intuitiva e estilizada para proporcionar uma boa experiência ao usuário. O CEP, por exemplo, será validado automaticamente para garantir sua existência e irá trazer cidade, bairro, logradouro e estado.
